// capabilities

Everything your stack needs.
Nothing it doesn't.

A complete autonomous systems coordination stack — sensor ingestion to operator dispatch — built as composable microservices you can run all at once or pick apart.

◈
Sensor Fusion

Kalman EKF multi-source fusion. Ingests ADS-B, AIS, satellite, MQTT, and custom adapters into a unified world model.

◉
AI Dispatch

ONNX-native inference. Mission classifier and risk scorer trained on real-world data. Pluggable model registry.

◎
Real-Time Fabric

WebSocket entity streaming, MQTT pub/sub, Redis-backed world state. Sub-second updates across all connected operators.

⬡
Mission Tasking

Full mission state machine. Tiered response, autonomous escalation, drone dispatch, human-in-the-loop approval workflows.

◇
Operator Console

Three role-optimized views: OPS (full-screen map), COMMAND (3-column situational), DEV (adapter and schema tooling).

○
Adapter SDK

First-class adapter framework. OpenSky, AIS, CelesTrak built in. Extend with any sensor, protocol, or data source.
